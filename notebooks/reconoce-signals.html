Reconocimiento de señales de tráfico¶

En este notebook se va a construir una red neuronal convolucional (CNN) para clasificar señales de tráfico.

El dataset a utilizar será el publicado en https://www.kaggle.com/meowmeowmeowmeowmeow/gtsrb-german-traffic-sign de la competición GTSRB - German Traffic Sign Recognition Benchmark. De la estructura del dataset utilizaremos los siguientes ficheros y directorios

  • Directorio train: Contiene un subdirectorio con el identificador de cada una de las clases. Dentro de cada subdirectorio hay un número variable de ejemplos de la clase con dimensiones variadas.
  • Fichero Train.csv: Incluye entre otras información de cada una de las fotos, su path, clase...

for i in range(classes):
    # Obtenemos el path a cada una de las carpetas de clase de la carpeta Train (de 0 a 42 en este caso)
    path = os.path.join(cur_path,'archive\Train',str(i))
    # Carga en la variable images el nombre de los ficheros del directorio de la variable path
    images = os.listdir(path)
    # Abre cada una de las imágenes, la redimensiona a 30x30, la convierte en un np array y la incluye 
    # en la lista de señales. También añade la label a la lista de labels.
    for a in images:
        try:
            image = Image.open(path + '\\' + a)
            image = image.resize((30,30))
            image = np.array(image)
            traffic_signs.append(image)
            labels.append(i)
        except:
            print("error")
# Convierte la lista de traffic_signs y labels a un np array para poder hacer el split train-test
traffic_signs = np.array(traffic_signs)
labels = np.array(labels)
print(traffic_signs.shape, labels.shape)

# Split train y test dataset con una proporcion 80-20
X_train, X_test, y_train, y_test = train_test_split(traffic_signs, labels, test_size=0.2, random_state=42)

# Codificar las labels mediante one hot encoding
y_train = to_categorical(y_train, 43)
y_test = to_categorical(y_test, 43)

# Construcción del modelo
model = Sequential()
# Aplicamos 2 capas de convolución de 32 kernels (filtros) con un tamaño de kernel 5x5
model.add(Conv2D(filters=32, kernel_size=(5,5), activation='relu', input_shape=X_train.shape[1:]))
model.add(Conv2D(filters=32, kernel_size=(5,5), activation='relu'))
# Esta capa va cogiendo trozos de 2x2 y retornando el valor mayor. Reduce el número de neuronas a la mitad
model.add(MaxPool2D(pool_size=(2, 2)))
# Pone un 25% de las entradas a 0 (para evitar overfiting). Las entradas no puestas a 0
# se escalan para que la suma de todas no cambie
model.add(Dropout(rate=0.25))
# Aplicamos 2 capas de convolución de 64 kernels (filtros) con un tamaño de kernel 3x3
model.add(Conv2D(filters=64, kernel_size=(3, 3), activation='relu'))
model.add(Conv2D(filters=64, kernel_size=(3, 3), activation='relu'))
model.add(MaxPool2D(pool_size=(2, 2)))
model.add(Dropout(rate=0.25))
# Aplana la entrada en un vector de una única dimensión
model.add(Flatten())
# Crea una capa dense que tiene como salida 256 neuronas
model.add(Dense(256, activation='relu'))
# Elimina la mitad
model.add(Dropout(rate=0.5))
# Crea una capa dense con 43 neuronas (tantas como número de clases)
# la activation softmax asegura que se eliminan las neuronas que aportan menos valor
model.add(Dense(43, activation='softmax'))
#Compilation of the model
model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])
# Se configuran 15 etapas para entrenar la red
epochs = 15
# Guarda en una variable el histórico de precision según epochs. Se utilizará mas adelante para
# mostrar una gráfica de la evolución
history = model.fit(X_train, y_train, batch_size=32, epochs=epochs, validation_data=(X_test, y_test))
# Se guarda el modelo entrenado con el nombre my_model.h5
model.save("my_model.h5")
